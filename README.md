--- Consumo de API com JavaScript (GET + POST)

Este projeto demonstra como consumir uma API pública usando JavaScript, aplicando:

fetch()

async/await

Tratamento de erros (try/catch)

Manipulação de respostas JSON

Requisições GET e POST

A API utilizada é a JSONPlaceholder.



--- Funcionalidades

GET: Carrega posts da API e exibe na página.

POST: Envia um novo post preenchido pelo usuário.

Interface simples em HTML + CSS + JS.
